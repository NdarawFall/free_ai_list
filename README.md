# Free AI List

Site React + Tailwind + GSAP pour lister des outils d'IA gratuits, stockes dans Supabase.

## Installation

```bash
npm install
npm run dev
```

## Variables d'environnement

Copiez `.env.example` vers `.env.local`, puis renseignez :

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Supabase

Le schema SQL est dans `supabase/schema.sql`.
